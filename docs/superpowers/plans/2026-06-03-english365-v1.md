# English365 V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the confirmed V1 static PWA for American English Trainer with 10 scenes, 200 sentence prompts, 20 conversations, localStorage progress, and Continue Learning.

**Architecture:** The app is a single-page static frontend. It uses global JavaScript modules loaded by `index.html`, a central store for state transitions, localStorage wrappers for persistence, and scene data packs registered through `data/index.js`. No `router.js` is created; views switch from store state.

**Tech Stack:** HTML, CSS, Vanilla JavaScript, localStorage, SpeechSynthesis, Web App Manifest, Service Worker.
